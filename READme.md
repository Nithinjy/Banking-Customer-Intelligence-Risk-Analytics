# 🏦 Banking Customer Intelligence & Risk Analytics Dashboard


## 📌 Project Overview

This project presents an end-to-end **Banking Customer Intelligence & Risk Analytics Dashboard** built using **Python** and **Streamlit**. The solution analyzes **100,000+ customer records** to uncover valuable insights into customer behavior, profitability, churn, loan risk, and cross-sell opportunities.

Using comprehensive **Exploratory Data Analysis (EDA)** and interactive dashboards, the project demonstrates how data-driven insights can support strategic business decisions in the banking industry.

---

## 🎯 Business Objectives

- Analyze customer demographics and banking behavior
- Identify churn and loan default risk factors
- Measure customer profitability
- Discover high-value customer segments
- Evaluate cross-sell opportunities
- Monitor banking KPIs through an interactive dashboard

# 💡 Key Insights

The analysis uncovered several actionable business insights:

- Identified **high-income customers maintaining relatively low account balances**, indicating potential investment and wealth management opportunities.
- Discovered customers with **strong credit scores but recent missed payments**, highlighting hidden credit risk.
- Identified **high-revenue yet low-profit customer segments**, suggesting opportunities for cost optimization.
- Found customer segments exhibiting elevated churn risk based on engagement and financial behavior.
- Revealed cross-sell opportunities across under-served customer groups.
- Observed strong relationships between digital engagement and customer activity.

---

# 📌 Business Recommendations

Based on the analysis:

- Improve customer retention through personalized engagement strategies.
- Strengthen early-warning systems for credit risk monitoring.
- Increase digital banking adoption among low-engagement customers.
- Prioritize high-value customer relationships.
- Launch targeted cross-selling campaigns.
- Continuously monitor business KPIs for proactive decision-making.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Banking-Customer-Intelligence-Dashboard.git
```

Move into the project directory

```bash
cd Banking-Customer-Intelligence-Dashboard
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the dashboard

```bash
streamlit run app.py
```
